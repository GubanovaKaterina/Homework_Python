Название:
Метод contains чувствителен к регистру, что противоречит документации.

Описание:
Метод contains реализован с использованием string.index(symbol), который выполняет регистрозависимый поиск. 
Это не соответствует интуитивному пониманию проверки наличия символа в строке.

Шаги для воспроизведения:
import pytest
from string_utils import StringUtils
string_utils = StringUtils()
@pytest.mark.positive
@pytest.mark.parametrize("input_str, symbol, expected", [
    ("House", "h", True),
])
def test_contains_positive(input_str, symbol, expected):
    assert string_utils.contains(input_str, symbol) == expected

Ожидаемый результат:
contains("House", "h") -> True

Фактический результат:
contains("House", "h") -> False
contains("House", "H") -> True

Серьезность: S2
Приоритет: P2

Возможные решения:
Изменить документацию: явно указать, что поиск регистрозависимый
Изменить реализацию: использовать регистронезависимый поиск
